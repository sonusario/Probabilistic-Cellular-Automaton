Probabilistic Cellular Automaton

Tabbed controls, variable neighbor masks, zoom/pan, and birth/death probability rules.

Controls

Run

Randomize preserves the current play/pause state. Step always pauses first.

Timing

A skip value of 1 calculates every step but draws every other generation.

Drawing

Left-drag draws. With line mode on, drag strokes fill gaps between sampled cells. The first touched cell decides whether the stroke paints live or dead.

Color gradients

Live cells use the death-rate gradient. Dead cells use the birth-rate gradient.

Global rule controls

The min/max range clamps every birth and death-rate control, including the two master sliders.

Neighbor definitions

Each 9×9 grid defines which surrounding cells count as neighbors. The center cell is the current cell and cannot be selected. Each mask must keep at least one active neighbor.
